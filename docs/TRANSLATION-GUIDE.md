# Þýðingaleiðbeiningar — Íslenskt byggingarefni úr ensku

> Þetta skjal skráir algengar villur við þýðingu byggingarefnis úr ensku á íslensku, og inniheldur
> þýðingartillögur fyrir hugtök sem vantar íslenskt heiti. Hugtök sem höfundur bjó til eru merkt
> með 🆕, en staðfest hugtök með ✅.

---

## 1. Algengar þýðingarvillur

### 1.1 „Prop" vs „Strut" vs „Brace" — Stoð, stífur, afstífing

Eitt algengasta vandamálið. Enska orðið „prop" getur þýtt ýmislegt:

| English | Rétt íslenska | **EKKI** þýða sem |
|---|---|---|
| Prop (vertical support) | **Stoð / Loftastoð** | ~~Stífur~~ (stífur er ská) |
| Strut / Diagonal brace | **Skástífur / Stífur** | ~~Stoð~~ (stoð er lóðrétt) |
| Push-pull prop / Aligning strut | **Mótaskástífur / Afstífing** | ~~Mótastoð~~ |
| Brace / Bracing | **Afstífing / Stífing** | ~~Stuðningur~~ (of almennt) |

**Regla:** *Stoð* stendur lóðrétt og ber þunga. *Stífur* gengur á ská og kemur í veg fyrir hreyfingu. *Afstífing* er kerfisbundið útstífingarverk.

### 1.2 „Tie" — Mörg mismunandi þýðingar

Orðið „tie" kemur reglulega fyrir í steypumóta- og pallafræðum en þýðist mjög mismunandi eftir samhengi:

| English | Íslenska | Samhengi |
|---|---|---|
| Formwork tie / Form tie | **Mótatengi / Mótaræður** | Heldur mótaflekaflötum saman |
| Tie rod | **Dregarateinn / Mótateinn** | Stangur sem gengur í gegn |
| Hexagonal tie | **Sexkanta ræður** | Ræður með sexkantformat |
| Wing tie | **Vængjaræður** | Ræður með vængjum |
| Wall tie (scaffolding) | **Veggfesting** | Festir vinnupall við vegg |
| Tie (rope/cable bracing) | **Togband / Strengjafesting** | Vindálagsmótvægi |

**Regla:** Athugaðu alltaf hvort um er að ræða steypumóta-„tie" (mótatengi/ræður) eða vinnupalla-„tie" (veggfesting). Þetta eru gjörólík hugtök!

### 1.3 „Panel" vs „Board" vs „Deck" — Fleki, borð, pallaborð

| English | Íslenska | Samhengi |
|---|---|---|
| Formwork panel | **Mótafleki / Fleki** | Steypumótsflötur |
| Scaffolding deck /  platform board | **Gólfborð / Pallaborð** | Gangborð vinnupalls |
| Toe board | **Táborð / Taborð** | Neðri kantvernd |
| Form board | **Mótaborð** | Timbur-/krossviðarborð á móti |
| Adjustment panel | **Stillanlegur mótafleki / Stillifleki** | Breytileg breidd |

**Regla:** *Fleki* er alltaf steypumótseining. *Borð* á vinnupalli (gangflötur). *Pallaborð* er stór borðeining.

### 1.4 „Jack" í mismunandi samhengi

| English | Íslenska | Samhengi |
|---|---|---|
| Base jack (scaffolding) | **Lappir** | Stillanlegur fótur á palli |
| Head jack (ID-15 / shoring) | **Hausar / Stoðhausar** | Efri handstilling |
| Hydraulic jack | **Tjakk / Dæla** | Vélbúnaður |
| Swivel base jack | **Veltilappir** | Snúningslappir á ójöfnu |
| Base jack (spindle) | **Spindill** | Kerfispallalappir |

**Algeng villa:** Að þýða öll „jack" sem „tjakk" — lappir og hausar eru **ekki** tjakkar!

### 1.5 „Corner" í steypumótum

| English | Íslenska | Lýsing |
|---|---|---|
| Inside corner | **Innhorn** | 90° innrihornseining |
| Outside corner | **Úthorn** | 90° ytrihornseining |
| Hinged corner | **Veltihorn** | Stillanleg hornseining |
| Corner filler | **Hornastillir / Hornastillar** | Fyllir bil í hornum |
| Corner piece | **Hornastykki** | Almennt |
| Corner plate | **Hornaplata** | Festiplata á horni |

### 1.6 „Clamp" vs „Coupler"

| English | Íslenska | Samhengi |
|---|---|---|
| Panel clamp | **Flekaklemma** | Hefur mótafleka saman |
| Aligning clamp | **Rettskeíðarklemma** | Stillir mót í lóð |
| Adjustable aligning clamp | **Stillanleg rettskeíðarklemma** | |
| Outside corner clamp | **Úthornaklemmur** | Festir úthorn |
| Scaffold coupler (right-angle) | **Föst klemma / Beinklemma** | 90° á röri |
| Swivel coupler | **Snúklemma** | Snýst á röri |
| Half coupler | **Hálfklemma** | Einhliða |

**Algeng villa:** Að nota „coupler" og „clamp" til skiptis — á íslensku er bæði *klemma* en samhengi skiptir máli.

### 1.7 „Frame" — Ekki alltaf „rammi"!

| English | Íslenska | Samhengi |
|---|---|---|
| Scaffolding frame | **Rammi** | Pallagerð |
| Door/window frame | **Karmi** | Byggingarsmíði |
| Storage frame / rack | **Grind / Rekki** | Geymsla |
| Bracing frame | **Tengistífa** | Afstífing |
| End frame | **Endarammi** | Endi á kerfi |

### 1.8 „Stripping" / „Striking" — Að slá frá

| English | Íslenska | **EKKI** þýða sem |
|---|---|---|
| Stripping / Striking (formwork) | **Slá frá / Mótarif** | ~~Rifa~~ (þ.e. rifa í sundur) |
| Erection / Forming | **Uppsláttur / Uppslátt** | ~~Uppbygging~~ (of almennt) |
| Early striking | **Snemmrif** | ~~Snemm riftun~~ |

**Regla:** „Rif" í steypumótasamhengi þýðir varlega losun mótanna — ekki eyðileggingu!

### 1.9 Kyn samsettrar orða (grammatík)

Íslensk samsett orð taka kyn síðasta samsetningarliðar:

| Orð | Kyn | Af hverju |
|---|---|---|
| Mótafleki | **kk.** | ← fleki (kk.) |
| Loftastoð | **kvk.** | ← stoð (kvk.) |
| Steypumót | **hk.** | ← mót (hk.) |
| Veggfesting | **kvk.** | ← festing (kvk.) |
| Skástífur | **kk.** | ← stífur (kk.) |
| Handriðsstoð | **kvk.** | ← stoð (kvk.) |

**Algeng villa:** Að höndla „mótafleka" eins og kvenkyns orð — *fleki* er karlkynsorð!

### 1.10 Falskir vinir (false friends)

| Enska | Röng þýðing | Rétt þýðing |
|---|---|---|
| Concrete (noun) | ~~Steinsteipa~~ | **Steypa** |
| Concrete (verb) | ~~Hörðna~~ | **Steypa / Að steypa** |
| Standard (scaffold upright) | ~~Staðall~~ | **Stoð / Uppistöðurör** |
| Ledger (horizontal member) | ~~Bókhald~~ | **Bjálki / Burðarrör** |
| Bearer | ~~Berandi~~ | **Burðarhluti / Burðarbiti** |
| Sole plate | ~~Botnplata~~ | **Fótplata / Sólarplata** |
| Scaffold | ~~Gálgi~~ | **Vinnupallur** |
| Release agent | ~~Losunarlík~~ | **Mótaolía / Losunarvökvi** |

## 4. Þýðingaraðferðafræði

### Hvernig við þýðum byggingarhugtök á íslensku

1. **Virkni fyrst** — Nefna hlutinn eftir því hvað hann *gerir* (t.d. „felligaffall" = fellir niður, gaffall-laga)
2. **Útlit í öðru lagi** — Ef orðið lýsir lögun (t.d. „C-krókur" = C-laga krókur, „sexkanta ræður")
3. **Samsett orð** — Íslenska notar samsett orð frekar en lýsingarorðafrasa:
   - ✅ *Mótaskástífur* (mót + ská + stífur)
   - ❌ ~~Skáhattur stífur fyrir mót~~
4. **Eignarfall í samsetningum** — Fyrri liður fer gjarnan í eignarfall:
   - Mót**a**fleki (mót → móta)
   - Loft**a**stoð (loft → lofta)
   - Vegg**ja**pallur (veggur → veggja)
5. **Varast beinar þýðingar** — Ekki þýða orð fyrir orð:
   - ❌ ~~Veggtengill~~ → ✅ **Veggfesting** (fyrir „wall tie")
   - ❌ ~~Standvinnuvettvangur~~ → ✅ **Vinnupallur** (fyrir „scaffold")
6. **Merkingarfræði skiptir máli** — Eitt enskt orð → mörg íslensk:
   - „Post" → stoð (stálstoð), pósta (handriðspósta), sýla (steypt)

### Nýyrðasmíð — Reglur

Þegar ekkert íslenskt orð er til:

1. Athuga fyrst: Er til sambærilegt orð á norsku/dönsku/sænsku sem hægt er að aðlaga?
2. Nota enskt lánorð **aðeins** ef það er nú þegar innrennt í málið (t.d. „krani" = crane)
3. Smíða samsett orð úr íslenskum stofnum sem lýsa virkni eða útliti
4. Athuga samræmi — ef svipuð vara er þegar þýdd, fylgja sama mynstri:
   - Ef „drop head" = „felligaffall" → drop + head → felli + gaffall ✓

## 5. Samantekt nýsmíða

| Nýtt orð | Byggt á | Vísar í |
|---|---|---|
| **HT-Borðhaus** | borð (table) + haus (head) | HT-Table Head |
| **C-Lyftukrókur** | lyftu- (lifting) + krókur (hook) | C-Hook / Crane Hook |
| **Staflagrind** | stafla (to stack) + grind (frame/rack) | Stacking Frame |
| **Endastykki** | enda- (end) + stykki (piece) | Terminal End |
| **Léttgerð** | létt (light) + gerð (make/type) | Lightweight |
| **Tvíbiti** | tví- (twin/double) + biti (beam) | TwixBeam |
| **Sveigibiti** | sveigi- (flex) + biti (beam) | FlexBeam |

Athugaðu — *Endastykki* er raunverulega til í almennu máli og hefur verið notað í öðru samhengi, en er hér beitt á vegatalma. Telst samt nýbeiting og er merkt sem nýsmíði.

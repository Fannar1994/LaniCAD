Vinnupallar – Scaffold Calculator Specification

This document describes the structure and calculations of the Vinnupallatafla með forsendum (Tvöföld handrið) Excel workbook for scaffolding (vinnupallar) and maps it to a React + TypeScript calculator. The Excel workbook is the single source of truth for all quantities and formulas. The code must mirror the Excel logic exactly.

1. Inputs per pallur (platform)

The sheet defines multiple platforms (pallar) as columns. In Excel there are columns for Pallur nr. 1–20 (or more). Below is the structure with the example values you provided for Pallur 1–4:

In the React calculator, you should expose these fields as numeric inputs per pallur (with support for all pallar in the original sheet).

2. Item list – Söluvrn / Leigunumer / Heiti & sample quantities

Each item row in Excel has:

  • Söluvrn – sales item number.
  • Leigunumer – rental item number.
  • Heiti – description.
  • Quantity per pallur (20+ columns).
  • Samtals – total quantity across all pallar.
  • Ein.verð – unit daily rental price.
  • Samtals – line total daily price (quantity × Ein.verð).

3. Totals and known formulas

3.1 Fermetrafjöldi (area)

The workbook has a row labelled Fermetrafjöldi: with one value per pallur and a total. In your example:

  • Pallur 1: 54 m²
  • Pallur 2: 54 m²
  • Pallur 3: 54 m²
  • Pallur 4: 54 m²
  • Total Fermetrar: 216 m²

The exact formula for Fermetrafjöldi must be taken from the Excel workbook (likely based on bay length and working/standing height). The code should:

  • Compute Fermetrafjöldi per pallur with the same formula as Excel.
  • Compute Fermetrar as the sum of all Fermetrafjöldi values.

3.2 Line totals and daily total

For each item row:

  • Samtals magn = sum of quantities across all pallar.
  • Línusamtals (dagleiga) = Samtals magn × Ein.verð.

Example checks from your data:

  • RAMMAR 2,0M: 50 × 19 = 950
  • GÓLFBORÐ 1,8M: 78 × 12 = 936
  • LAPPIR 50CM: 284 × 6 = 1.704

The Samtals dagleiga með virðisaukaskatti is the sum of all line totals (including VAT), in your example:

  • Samtals dagleiga m/vsk: 5.654

3.3 Monthly rent

The Mánaðarleiga (30d) m/vsk is computed by multiplying the daily total with 30:

  • Mánaðarleiga (30d) m/vsk = 5.654 × 30 = 169.620

In code: 

monthlyTotalWithVat = dailyTotalWithVat * 30;

4. Mapping to React + TypeScript

The VS Code agent should implement the following (or similar) types and functions: 

type PallurInput = {
  pallurNr: number;
  einingafjoldi: number;
  fjoldi2mHaeda: number;
  fjoldi0_7mHaeda: number;
  haestaStandhaed: number;
  mestaVinnuhaed: number;
  fjoldiEndalokana: number;
};

type ItemRow = {
  salesCode: string;
  rentalCode: string;
  name: string;
  quantitiesPerPallur: number[];
  totalQuantity: number;
  unitPrice: number;
  lineTotal: number;
};

type ScaffoldResult = {
  items: ItemRow[];
  fermetrafjoldiPerPallur: number[];
  totalFermetrar: number;
  dailyTotalWithVat: number;
  monthlyTotalWithVat: number;
};
function calculateScaffold(pallar: PallurInput[]): ScaffoldResult {
  // 1. For each PallurInput, compute Fermetrafjöldi using the same formula
  //    as in the Excel workbook.
  // 2. For each item, compute quantitiesPerPallur from the pallar inputs
  //    using the same formulas as in Excel.
  // 3. totalQuantity = sum(quantitiesPerPallur).
  // 4. lineTotal = totalQuantity * unitPrice.
  // 5. dailyTotalWithVat = sum(lineTotal for all items).
  // 6. monthlyTotalWithVat = dailyTotalWithVat * 30.
}

The React component should be implemented as ScaffoldDoubleGuardrailCalculator.tsx using React 18 functional components and hooks, exposing:

  • Form inputs for all PallurInput fields per pallur.
  • A result table for the item rows and totals as described above.

Rasto/Takko – Steypumót Calculator Specification

This spec defines a React + TypeScript calculator for Rasto/Takko steypumót based on the Excel sheet “Tilboð í Leigu á Rasto/Takko steypumótum”.

  • Only the items listed in this file may be used.
  • No new panel sizes or accessories may be invented in code.
  • All areas and prices come from the “m2” and “Dagleiga m/vsk” columns.

Represent all of the above items in TypeScript as a static array: 

type RastoTakkoItem = {
  erlVoruNr: string | null;
  supplierCode: string;
  name: string;
  group: "Grunneiningar" | "Samtengihlutir" | "Aukahlutir";
  areaM2: number;            // 0 if not applicable
  dailyPriceWithVat: number; // Dagleiga m/vsk
};

2. Inputs

From the offer header:

  • Fjöldi fermetra – total required formwork area.
  • Hæð Steypumóta – formwork height (e.g. 120 cm).
  • Fjöldi lengdarmetra í tvöföldun – length of wall that is double-sided.
Field Type Unit Description
totalAreaM2 number m² Fjöldi fermetra (total shuttered area).
formworkHeightCm number cm Hæð steypumóta (e.g. 120 or 300).
doubleLengthM number m Fjöldi lengdarmetra í tvöföldun – length of double-sided wall.
rentalDays number days Chargeable rental period.
vatRate number - VAT as fraction (e.g. 0.24 for 24%). 
type RastoTakkoInputs = {
  totalAreaM2: number;
  formworkHeightCm: number; // 120 or 300 typically
  doubleLengthM: number;    // length that is double-sided
  rentalDays: number;
  vatRate: number;
};

3. Configuration factors (you control these)

Accessories and connectors are driven by simple factors based on total area, total panel count and doubleLengthM. These factors are not invented in code – you set them. 

type RastoTakkoFactors = {
  connectorsPerPanel: number;   // e.g. Flekaboltar per panel
  tiesPerM2: number;            // Mótarær per m²
  bracesPer10mDouble: number;   // R-skástífur tvöfaldar per 10 m of doubleLengthM
  workDecksPer10m: number;      // Rasto vinnupallaknekti per 10 m (if you want)
  handrailPostsPer10m: number;  // TK-Handriðspóstar per 10 m (if you want)
};

4. Panel selection logic (Grunneiningar only)

Use only items with group = "Grunneiningar" and areaM2 > 0 as panels. The height selection is based on formworkHeightCm:

  • If formworkHeightCm = 300: use items whose name contains x300.
  • If formworkHeightCm = 120: use items whose name contains x120.
function selectPanelCandidates(
  items: RastoTakkoItem[],
  formworkHeightCm: number
): RastoTakkoItem[] {
  const tag = formworkHeightCm === 300 ? "x300" : "x120";

  return items
    .filter(i =>
      i.group === "Grunneiningar" &&
      i.areaM2 > 0 &&
      i.name.includes(tag)
    )
    .sort((a, b) => b.areaM2 - a.areaM2); // largest area first
}

Greedy coverage of totalAreaM2 using only these panels: 

type PanelQuantity = { erlVoruNr: string | null; quantity: number; };

function computePanelQuantities(
  panels: RastoTakkoItem[],
  totalAreaM2: number
): PanelQuantity[] {
  let remaining = totalAreaM2;
  const map = new Map<string | null, number>();

  for (let i = 0; i < panels.length; i++) {
    const p = panels[i];
    const area = p.areaM2;
    if (area <= 0) continue;

    if (i < panels.length - 1) {
      const count = Math.floor(remaining / area);
      if (count > 0) {
        const prev = map.get(p.erlVoruNr) || 0;
        map.set(p.erlVoruNr, prev + count);
        remaining -= count * area;
      }
    } else {
      // last (smallest) panel: if anything remains, add 1
      if (remaining > 0) {
        const prev = map.get(p.erlVoruNr) || 0;
        map.set(p.erlVoruNr, prev + 1);
        remaining = 0;
      }
    }
  }

  const result: PanelQuantity[] = [];
  map.forEach((q, code) => result.push({ erlVoruNr: code, quantity: q }));
  return result;
}

This guarantees panel area sum ≥ totalAreaM2 using only the real panel types from your offer.

5. Accessories & connectors quantities

Once panel quantities are known, derive accessories according to RastoTakkoFactors. 

type ItemQuantity = { erlVoruNr: string | null; quantity: number; };

function computeAccessoryQuantities(
  factors: RastoTakkoFactors,
  items: RastoTakkoItem[],
  panelQuantities: PanelQuantity[],
  inputs: RastoTakkoInputs
): ItemQuantity[] {
  const totalPanels = panelQuantities
    .reduce((sum, p) => sum + p.quantity, 0);
  const totalArea = inputs.totalAreaM2;
  const doubleLength = inputs.doubleLengthM;

  const result: ItemQuantity[] = [];

  // Example mappings (you will tune which items use which factor):

  // 1) Connectors per panel (RT-Flekaboltar + RT-Flekarær etc.)
  const connectorsPerPanel = factors.connectorsPerPanel;
  if (connectorsPerPanel > 0) {
    const totalConnectors = Math.ceil(connectorsPerPanel * totalPanels);
    // RT-Flekaboltar:
    result.push({ erlVoruNr: "01-MÓT-HM21-264", quantity: totalConnectors });
    // RT-Flekarær:
    result.push({ erlVoruNr: "01-MÓT-HM21-566", quantity: totalConnectors });
  }

  // 2) Ties per m² (Mótarær DW 15)
  const tiesPerM2 = factors.tiesPerM2;
  if (tiesPerM2 > 0) {
    const ties = Math.ceil(tiesPerM2 * totalArea);
    result.push({ erlVoruNr: "01-MÓT-AH21-332", quantity: ties });
  }

  // 3) Braces per 10 m double-sided (R-skástífur tvöfaldar)
  const bracesPer10m = factors.bracesPer10mDouble;
  if (bracesPer10m > 0 && doubleLength > 0) {
    const braces = Math.ceil((doubleLength / 10) * bracesPer10m);
    result.push({ erlVoruNr: "01-MÓT-HM21-381", quantity: braces });
  }

  // Add more rules here for other Aukahlutir if you want, using the same pattern.

  return result;
}

The important part: all Erl.Vörunr. referenced in this function must come from the tables in this spec. No other item codes are allowed.

6. Pricing

For any item with quantity q and dailyPriceWithVat price: 

lineDailyWithVat   = q * price;
lineDailyExVat     = lineDailyWithVat / (1 + vatRate);
lineTotalWithVat   = lineDailyWithVat * rentalDays;
lineTotalExVat     = lineTotalWithVat / (1 + vatRate);

Totals: 

totalDailyWithVat = sum(lineDailyWithVat for all items);
totalDailyExVat   = sum(lineDailyExVat for all items);
totalWithVat      = sum(lineTotalWithVat for all items);
totalExVat        = sum(lineTotalExVat for all items);
type RastoTakkoItemResult = {
  erlVoruNr: string | null;
  supplierCode: string;
  name: string;
  group: "Grunneiningar" | "Samtengihlutir" | "Aukahlutir";
  quantity: number;
  dailyPriceWithVat: number;
  lineDailyWithVat: number;
  lineDailyExVat: number;
  lineTotalWithVat: number;
  lineTotalExVat: number;
};

type RastoTakkoTotals = {
  totalAreaM2: number;
  totalDailyWithVat: number;
  totalDailyExVat: number;
  totalWithVat: number;
  totalExVat: number;
};

type RastoTakkoResult = {
  inputs: RastoTakkoInputs;
  items: RastoTakkoItemResult[];
  totals: RastoTakkoTotals;
};

7. Core calculation function

function calculateRastoTakkoOffer(
  items: RastoTakkoItem[],
  inputs: RastoTakkoInputs,
  factors: RastoTakkoFactors
): RastoTakkoResult {
  // 1) Select panel candidates by formworkHeightCm (x300 or x120).
  const panelCandidates = selectPanelCandidates(items, inputs.formworkHeightCm);

  // 2) Compute panel quantities to cover inputs.totalAreaM2.
  const panelQuantities = computePanelQuantities(
    panelCandidates,
    inputs.totalAreaM2
  );

  // 3) Compute accessory & connector quantities from factors.
  const accessoryQuantities = computeAccessoryQuantities(
    factors,
    items,
    panelQuantities,
    inputs
  );

  // 4) Combine all quantities into one map of erlVoruNr -> quantity.
  const quantitiesMap = new Map<string | null, number>();
  function addList(list: { erlVoruNr: string | null; quantity: number; }[]) {
    for (const it of list) {
      if (!it.erlVoruNr) continue;
      const prev = quantitiesMap.get(it.erlVoruNr) || 0;
      quantitiesMap.set(it.erlVoruNr, prev + it.quantity);
    }
  }
  addList(panelQuantities);
  addList(accessoryQuantities);

  // 5) Build item result rows with pricing.
  const resultItems: RastoTakkoItemResult[] = [];
  for (const item of items) {
    const q = quantitiesMap.get(item.erlVoruNr) || 0;
    if (q <= 0) continue;

    const lineDailyWithVat   = q * item.dailyPriceWithVat;
    const lineDailyExVat     = lineDailyWithVat / (1 + inputs.vatRate);
    const lineTotalWithVat   = lineDailyWithVat * inputs.rentalDays;
    const lineTotalExVat     = lineTotalWithVat / (1 + inputs.vatRate);

    resultItems.push({
      erlVoruNr: item.erlVoruNr,
      supplierCode: item.supplierCode,
      name: item.name,
      group: item.group,
      quantity: q,
      dailyPriceWithVat: item.dailyPriceWithVat,
      lineDailyWithVat,
      lineDailyExVat,
      lineTotalWithVat,
      lineTotalExVat
    });
  }

  // 6) Totals
  let totalDailyWithVat = 0;
  let totalDailyExVat = 0;
  let totalWithVat = 0;
  let totalExVat = 0;

  for (const r of resultItems) {
    totalDailyWithVat += r.lineDailyWithVat;
    totalDailyExVat  += r.lineDailyExVat;
    totalWithVat     += r.lineTotalWithVat;
    totalExVat       += r.lineTotalExVat;
  }

  const totals: RastoTakkoTotals = {
    totalAreaM2: inputs.totalAreaM2,
    totalDailyWithVat,
    totalDailyExVat,
    totalWithVat,
    totalExVat
  };

  return {
    inputs,
    items: resultItems,
    totals
  };
}

8. React component

Implement RastoTakkoFormworkCalculator.tsx:

  • React 18 + TypeScript, functional component with hooks.
  • Inputs:
    • totalAreaM2
    • formworkHeightCm (default 120)
    • doubleLengthM
    • rentalDays
    • vatRate
    • RastoTakkoFactors (either fixed or editable).
  • On any change, call calculateRastoTakkoOffer and render:
    • Table of all selected items (quantity, daily, total).
    • Summary with totals ex./inc. VAT and total m².
  • No item may appear that is not defined in section 1 of this spec.
